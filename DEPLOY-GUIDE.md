# Putting Metal Clicker on GitHub Pages

This guide gets your game a real web link like
`https://yourname.github.io/metal-clicker/` that you can open in a browser.
No coding needed, just clicking around the GitHub website.

GitHub Pages is free, and it only hosts simple websites and files (like this
game), so it's a legitimate place to put it.

---

## Before you start

- You need the game file: **metal-clicker.html**
- You need an email address you can check (to confirm your account)
- It takes about 10–15 minutes the first time

---

## Step 1 — Make a GitHub account

1. Go to **https://github.com**
2. Click **Sign up**
3. Enter your email, pick a username, and a password.
   - Your username becomes part of your link, so pick something clean.
     If your username is `coolsmith`, your link will start with
     `https://coolsmith.github.io/...`
4. Verify your email when GitHub sends you a confirmation.

> If you're under 13, GitHub's rules don't allow an account. In that case,
> ask a parent or teacher to help set this up, or just share the file directly
> (see "If you can't use GitHub" at the bottom).

---

## Step 2 — Create a repository (a project folder)

1. Once logged in, click the **+** in the top-right corner, then
   **New repository**.
2. **Repository name:** type `metal-clicker`
3. Set it to **Public** (Pages needs this on a free account).
4. Tick **Add a README file**.
5. Click **Create repository**.

---

## Step 3 — Upload the game file

1. On your new repository page, click **Add file** then **Upload files**.
2. Drag **metal-clicker.html** into the box (or click "choose your files").
3. **Important:** rename the file to **index.html** so it loads
   automatically. You can do this before uploading, or after (see the tip
   below).
4. Scroll down and click **Commit changes**.

> **Tip — why rename to index.html?**
> A web page named `index.html` is the one a site shows by default. If you
> keep the name `metal-clicker.html`, your link just needs the full name on
> the end: `https://yourname.github.io/metal-clicker/metal-clicker.html`.
> Renaming to `index.html` gives you the cleaner link:
> `https://yourname.github.io/metal-clicker/`

---

## Step 4 — Turn on GitHub Pages

1. In your repository, click **Settings** (top menu).
2. In the left sidebar, click **Pages**.
3. Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
4. Under **Branch**, pick **main** and the folder **/(root)**, then click
   **Save**.
5. Wait 1–3 minutes. Refresh the page. A green banner will show your live
   link, something like:
   `https://yourname.github.io/metal-clicker/`

That link is your game. Open it in any browser.

---

## Step 5 — Updating the game later

When you want to put a newer version up:

1. Go to your repository.
2. Click your `index.html` file, then the pencil (Edit) icon — or delete it
   and upload the new file with the same name.
3. **Commit changes.**
4. Wait a minute or two; the live link updates automatically.

---

## About using it at school (read this)

A working link does **not** guarantee it opens at school. The NSW Department
of Education runs a web filter on school devices and networks, and it often
blocks games and unfamiliar sites no matter how the link was made.

The honest, safe way to use it at school:

- **Ask a teacher first.** Show them the link and ask if it can be allowed.
  Schools have a process for staff to approve a specific site. Going this
  route won't get your account flagged.
- Trying to find links specifically to dodge the filter can get your school
  account in trouble, so it's not worth it.

## One feature note

The **shared leaderboard** only works while the game runs inside Claude,
because it uses Claude's storage to share scores. On GitHub Pages everything
else works perfectly — clicking, upgrades, anvil skins, achievements, and
saving through the Export/Import codes in Settings — but the cross-player
leaderboard won't connect there.

## If you can't use GitHub

The game is a single self-contained file. You can just send
**metal-clicker.html** to someone (or to yourself) and double-click it to play.
No website or link required.
